# Yazi


## Installation
https://yazi-rs.github.io/docs/installation/#cargo

```bash

## avoir la version la plus recente de rustup (Rust Toolchain)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup update

# install from crates.io
cargo install --locked
```
